# Morse Practice
A free open-source web app to help you practice Morse code in a variety of ways, including write and read practices, along with output modes like flashlight, audio, or text.

# Supported Options
Practice Types:
- Write: Practice writing morse code by translating text to dots and dashes.
- Read: Practice reading morse code by translating dots and dashes to text.

Output Modes:
- Flashlight: Use the on screen flashlight to read Morse code.
- Audio: Listen to Morse code through audio beeps.
- Text: Display Morse code as text characters (dots and dashes).

Content Modes:
- Jumble: Practice with random letters.
- Words: Practice with random 5-letter words.
- Letters: Practice with the full alphabet.

Answer Modes:
- Word: Answer with full words.
- Letter: Answer with single letters, getting immediate feedback.
- Interpret: Interpret the morse code into dots and dashes.
- Static: Do it in your head mode, no answers given.
- Button (Write only): Tap a button to input morse code.
